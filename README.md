impexPlugin
===========

Simple impex syntax colorer for Eclipse 3.5+ Java 1.7

1) Download impexPlugin_1.0.2.jar from impexPluginUpdateSite/plugins/

2) Put the file in Eclipse/plugins

3) Restart

4) Enjoy
